boolval() Returns the boolean value(True or False) of a defined Variable
empty() Used to check if a defined variable is empty i.e. if it has an assigned value
gettype() Returns the data type of a variable (NULL, Boolean, Integer, String etc...)
intval() Returns the integer value of variables
is_array() Determines if a variable is an array or not
is_bool() Checks if a variable is Boolean
is_callable() Used to check if the data in a variable can be called as a function
is_double() or is_float() Checks if the value in a variable is a float type(decimal)
is_int() or is_integer() Determines if the data in a variable is an Integer
is_object() Checks if a variable is an object or not
is_string() Checks if a variable is a type String or not
print_r() Functions like the "echo" statement. Prints or outputs data in a human-readable format
settype() Converts variables from one form of type to another. for example:
$x = "49"; // string
settype($x, "integer"); // $x is now integer
var_dump() Dumps information like; type and value, of different variables. It functions like an advanced version of the print_r()
